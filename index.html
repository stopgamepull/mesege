1. What is EncryptedMessage?
EncryptedMessage is a service for creating Encrypted notes that can only be opened by the owner of the wallet for which the note was created.
2. How does EncryptedMessage work?
You create a note, send a link to the recipient. Only the owner of the recipient's wallet can decrypt the note.
3. Is it safe to use EncryptedMessage?
Yes, EncryptedMessage uses encryption and doesn't store notes after they're read. However, we don't recommend sending extremely confidential information through the service.
4. How long are unread notes stored?
Notes are stored on the server until they're first opened, but no longer than 7 days (unless a different period is specified).
5. Is it possible to recover a deleted note?
No, after deletion the note is permanently removed from the server and cannot be recovered.
6. Do I need to register to use EncryptedMessage?
No, EncryptedMessage doesn't require registration. The service is completely anonymous.
